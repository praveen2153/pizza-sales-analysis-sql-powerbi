# 🍕 Pizza Sales Analysis using SQL and Power BI

## 📌 Project Overview

This project performs end-to-end sales analysis on pizza order data using MySQL, SQL, and Power BI. The objective is to derive meaningful business insights and create an interactive dashboard that helps monitor sales performance and customer preferences.

The workflow includes importing raw data into MySQL, extracting insights through SQL queries, connecting the database with Power BI, performing data transformation, and building an interactive dashboard.

---

## 🎯 Problem Statement

The goal of this project is to analyze pizza sales data and identify key performance indicators and business trends to support data-driven decision making.

# 📊 KPI Requirements

The following KPIs were analyzed:

### 1. Total Revenue
Sum of the total sales generated.

### 2. Average Order Value
Average amount spent per order.

### 3. Total Pizzas Sold
Total quantity of pizzas sold.

### 4. Total Orders
Total number of customer orders.

### 5. Average Pizzas Per Order
Average quantity of pizzas purchased per order.
